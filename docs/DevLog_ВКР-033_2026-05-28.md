### Ключевые решения

**`views.py`:**
- Импорт `Service`, `News` из `.models`
- `HOME_SERVICES_COUNT = 3`, `HOME_NEWS_COUNT = 3` — именованные константы, не literals
- `Service.objects.filter(is_active=True).order_by('title')[:HOME_SERVICES_COUNT]`
- `News.objects.filter(published_at__isnull=False).order_by('-published_at').select_related('author')[:HOME_NEWS_COUNT]`

**`home.html`:**
- `{% if request.user.is_authenticated %}` — условный CTA без обращения к бэкенду
- `{{ service.description|truncatewords:20 }}` — безопасное усечение описания
- Все иконки — inline SVG Heroicons (outline, 24–28px), `aria-hidden="true"`
- `role="list"` на `<ul>` для screen readers
- `aria-labelledby` на каждой `<section>`
- Empty state: `role="status"`, CTA → contacts

**`style.css`:**
- Только CSS-переменные, ноль raw HEX или px-значений
- `.hero` фон — gradient из `--color-primary-light` → `--color-white`
- `auto-fit minmax()` — responsive grid без медиазапросов
