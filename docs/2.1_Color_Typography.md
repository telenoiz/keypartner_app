# 2.1 Цветовая схема и типографика

## 2.1.1 Цветовая схема

Цветовая палитра построена на основе семантических ролей. Все пары фон/текст проверены на соответствие WCAG 2.1 AA (контраст ≥ 4.5:1 для текста, ≥ 3:1 для крупных элементов).

### Основные цвета (CSS-переменные)

| Роль | Переменная | HEX | Применение |
|------|-----------|-----|-----------|
| Primary | `--color-primary` | `#2563EB` | Основные кнопки, активные ссылки, акценты |
| Primary Dark | `--color-primary-dark` | `#1D4ED8` | Hover-состояние primary-элементов |
| Primary Light | `--color-primary-light` | `#DBEAFE` | Фон выделенных блоков, badges |
| Success | `--color-success` | `#16A34A` | Статус «Выполнено», подтверждения |
| Warning | `--color-warning` | `#D97706` | Статус «В работе», предупреждения |
| Danger | `--color-danger` | `#DC2626` | Ошибки, статус «Отклонено», деструктивные действия |
| Neutral 900 | `--color-neutral-900` | `#111827` | Основной текст |
| Neutral 600 | `--color-neutral-600` | `#4B5563` | Вторичный текст, подписи |
| Neutral 300 | `--color-neutral-300` | `#D1D5DB` | Разделители, рамки |
| Neutral 100 | `--color-neutral-100` | `#F3F4F6` | Фон карточек, таблиц |
| White | `--color-white` | `#FFFFFF` | Основной фон страниц |

### Семантические статусы заявок

| Статус | Цвет метки | Фон бейджа | HEX текст / фон |
|--------|-----------|------------|-----------------|
| Новая | `#1D4ED8` | `#DBEAFE` | синий |
| В работе | `#92400E` | `#FEF3C7` | янтарный |
| Ожидает ответа | `#5B21B6` | `#EDE9FE` | фиолетовый |
| Выполнена | `#166534` | `#DCFCE7` | зелёный |
| Отклонена | `#991B1B` | `#FEE2E2` | красный |

### Доступность (Accessibility)

- Контраст основного текста (`#111827` на `#FFFFFF`): **16.7:1** — AAA ✓
- Контраст primary-кнопки (`#FFFFFF` на `#2563EB`): **5.3:1** — AA ✓
- Контраст success (`#FFFFFF` на `#16A34A`): **4.6:1** — AA ✓
- Контраст danger (`#FFFFFF` на `#DC2626`): **4.7:1** — AA ✓

## 2.1.2 Типографика

### Шрифт: Inter

**Inter** (Rasmus Andersson, Google Fonts, open source) выбран как основной шрифт интерфейса по следующим критериям:

- Оптимизирован для экранного чтения при малых кеглях
- Поддерживает кириллицу (включая все необходимые символы русского языка)
- Нейтральный, деловой характер — соответствует B2B-продукту
- Широкая поддержка браузеров, CDN-доступность через Google Fonts
- Variable Font — один файл покрывает все начертания (weight 100–900)

### Масштаб шрифтов

| Уровень | Переменная | Размер | Weight | Line-height | Применение |
|---------|-----------|--------|--------|-------------|-----------|
| Display | `--text-display` | 30px | 700 | 1.2 | Заголовок страницы (H1) |
| H2 | `--text-h2` | 24px | 600 | 1.3 | Разделы, модальные окна |
| H3 | `--text-h3` | 20px | 600 | 1.4 | Подразделы, заголовки карточек |
| H4 | `--text-h4` | 16px | 600 | 1.4 | Метки секций внутри форм |
| Body | `--text-body` | 14px | 400 | 1.6 | Основной текст интерфейса |
| Small | `--text-small` | 12px | 400 | 1.5 | Подписи, мета-информация |
| Caption | `--text-caption` | 11px | 500 | 1.4 | Бейджи, статусы, метки таблиц |

### Подключение (Django / HTML-шаблон)

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap"
      rel="stylesheet">
```

```css
:root {
  --font-base: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
}

body {
  font-family: var(--font-base);
  font-size: 14px;
  line-height: 1.6;
  color: var(--color-neutral-900);
  background-color: var(--color-white);
}
```

## 2.1.3 Иконки

Для системных иконок (навигация, действия, статусы) используется **Heroicons** (Tailwind Labs, MIT):
- Стиль: outline (24px) для интерфейса, solid (20px) для кнопок и бейджей
- Подключение: SVG-спрайт или inline SVG в шаблонах Django
