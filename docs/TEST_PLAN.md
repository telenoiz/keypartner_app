**Дата:** 04.06.2026  
**Версия:** 1.0

---

## 1. Методы тестирования

| Метод | Описание |
|---|---|
| Позитивное | Проверка штатных сценариев при корректных входных данных |
| Негативное | Проверка поведения при некорректных данных и граничных значениях |
| Функциональное | Проверка соответствия функций требованиям F01–F14 |
| Регрессионное | Проверка что исправление одного дефекта не нарушает другие функции |
