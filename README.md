# Веб-сервис для централизованного управления заявками клиентов

## О компании

**ООО «Ки Партнер ИТ»** — организация, специализирующаяся в области информационных технологий и автоматизации бизнес-процессов. Предприятие осуществляет деятельность по разработке и внедрению программных решений для корпоративных заказчиков.

## О проекте

Данный веб-сервис разрабатывается в рамках **преддипломной практики** и является основой для выполнения **выпускной квалификационной работы (ВКР)**.

### Тема ВКР

> *«Автоматизация процесса взаимодействия с клиентами в ООО "Ки Партнер ИТ" с использованием веб-сервиса для централизованного управления заявками»*

> ⚠️ **Тема находится на согласовании и может быть уточнена.**

### Рассматриваемый бизнес-процесс

> *«Взаимодействие с клиентами и централизованное управление заявками»*

### Планируемый функционал

- Приём, обработка и учёт заявок от клиентов
- Роли: Администратор, Менеджер (Сотрудник), Клиент (Пользователь)

### Ответственное подразделение

> Отдел информационных технологий / Департамент разработки ООО «Ки Партнер ИТ»
